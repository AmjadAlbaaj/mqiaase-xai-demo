## Getting Started

### Prerequisites
- Python 3.8 or higher
- Required dependencies (see `requirements.txt`)

### Installation
```bash
git clone https://github.com/AmjadAlbaaj/mqiaase-xai-demo.git
cd mqiaase-xai-demo
pip install -r requirements.txt
```

### Quick Start
```python
from mqiaase import AdaptiveSimulationEngine

# Initialize the engine
engine = AdaptiveSimulationEngine()

# Run your adaptive simulation
results = engine.simulate(config={
    'fidelity_level': 'adaptive',
    'explain_decisions': True
})

# Access results with full explainability
print(results.explanation)
print(results.accuracy)
```
